# 🚀 Personal Portfolio Website

A modern, responsive portfolio website showcasing my journey as a Software Engineer at ServiceNow. Built with clean HTML, CSS, and JavaScript with smooth animations and mobile-optimized interactions.

## ✨ Features

### 🎨 **Modern Design**
- Clean, professional layout with cyan accent colors
- Responsive design that works on all devices
- Dark theme-ready styling architecture
- Smooth hover effects and micro-interactions

### 📱 **Mobile-First Experience**
- Touch-optimized animations and interactions
- Tap ripple effects for visual feedback
- Smooth scroll animations optimized for mobile
- Performance-optimized with `requestAnimationFrame`

### ⚡ **Interactive Elements**
- Typing animation for subtitle text
- Scroll progress indicator
- Smooth scrolling navigation
- Hover effects on cards, buttons, and social icons
- Floating profile image animation

### 🔧 **Performance & Accessibility**
- Optimized animations with `will-change` properties
- Respects user's motion preferences (`prefers-reduced-motion`)
- Passive event listeners for better performance
- Font smoothing and optimized rendering

## 🛠️ Tech Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with CSS Grid/Flexbox
- **Vanilla JavaScript** - Custom animations and interactions
- **Bootstrap 5** - Responsive grid system
- **Font Awesome** - Icon library
- **WOW.js** - Scroll reveal animations
- **GLightbox** - Lightbox for project videos

## 🚀 Getting Started

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/mrishav/personal-website.git
   cd personal-website
   ```

2. **Open in browser**
   ```bash
   # Using Python (if installed)
   python -m http.server 8000
   
   # Using Node.js (if installed)
   npx serve .
   
   # Or simply open index.html in your browser
   open index.html
   ```

3. **View the website**
   Navigate to `http://localhost:8000` or open `index.html` directly

### Customization

1. **Update Content**
   - Edit `index.html` to update experience, projects, and personal information
   - Replace images in `assets/img/` with your own

2. **Modify Styling**
   - Primary styles are in `assets/css/main.css`
   - Color scheme can be updated by changing CSS custom properties
   - Cyan accent color: `#64ffda`

3. **Add New Sections**
   - Follow the existing HTML structure
   - Add corresponding CSS classes for animations
   - Update JavaScript observers if needed
